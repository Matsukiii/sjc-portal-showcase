# Public Information Office Management Portal (Showcase)

🚀 **Live Deployment:** [sjc.gov.ph](https://sjc.gov.ph)

*Note: This repository serves as a portfolio showcase. The source code is proprietary government property and is kept private for security and confidentiality compliance. I am happy to discuss the architecture, database design, and development process in an interview setting.*

---

## 📖 Project Overview
A full-stack web-based management portal and public-facing website developed for a local government Public Information Office. This system was built to digitize records, streamline public information dissemination, and automate internal data workflows. 

I architected and developed this system from scratch (0 → 1), managing the complete lifecycle from initial routing and database design to server deployment.

## 🛠️ Tech Stack
*   **Backend:** PHP, Laravel
*   **Database:** MySQL
*   **Frontend:** Blade Templating, HTML5, CSS3
*   **Environment & Deployment:** XAMPP, Batch Scripting

## ✨ Key Features & Technical Highlights
*   **Full-Stack Architecture:** Designed and implemented the complete MVC architecture using Laravel, ensuring clean separation of concerns and scalable code.
*   **Automated Data Processing:** Engineered complex data integration features, including building automated functions to seamlessly import, parse, and store data from Excel spreadsheets directly into the MySQL database.
*   **UI/UX Implementation:** Developed a responsive and accessible frontend, including a dynamic landing page, modular headers, and optimized footers for government-standard public access.
*   **Database Management:** Handled all database migrations, relational schema design, and secure data querying.
